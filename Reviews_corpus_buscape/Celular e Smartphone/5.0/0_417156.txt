O legal desse smartphone é a total compatibilidade com o Sistema Operacional Windows 8, este smartphone vem com o mais moderno sistema operacional da Microsoft para smartphones, o Window Phone 8 que vem com uma grande novidade, o NFC (Near Field Communication) que permite a troca de dados entre dispositivos que tem esta tecnologia, necessitando apenas aproximar o smartphone a outro dispositivo com a mesmo tecnologia, alguns recursos já disponíveis nesse smartphone usando esta tecnologia é o compartilhamento de arquivos e pagamentos de despesas sem necessidade de cartão de credito ou debito.
