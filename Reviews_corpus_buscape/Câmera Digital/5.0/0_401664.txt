Não é uma câmera profissional; mas é uma câmera muito boa, tira fotos com qualidade, é leve e satisfez minha necessidade de uma câmera portátil para o dia-a-dia. O acesso às funções via menus é menos fácil do que se fosse um acesso via botão externo; embora isso não prejudique muito a funcionalidade, é uma característica aquém das outras do produto.
